## API:
- 'GET', '/', - приветственный текст 
- 'GET', '/create' - пример создания пользователя, взять id и можно его переддать в нижестоящие запросы 
- 'GET', '/find?id=<id>' - пример поиска по id
- 'GET', '/delete?id=<id>' - пример удаления по id
