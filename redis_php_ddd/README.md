## Инструкция по конфигурации приложения

### Конфигурация инфраструктуры

Конфигурационный файл инфраструктуры (.env) находится в корне проекта:

``` /redis_php_ddd/.env ```

В нем указаны порты приложения и хранилищ.

### Конфигурация приложения

Конфигурационный файл приложения (.env) находится на уровне с точкой входа в PHP-приложение:

``` /redis_php_ddd/app/.env ```

В нем указывается используемое хранилище и конфигурационный файл DI-контейнера.

### Конфигурация DI-контейнера

Конфигурационный DI-контейнера (di-config.php) находится на уровне с точкой входа в PHP-приложение:

``` /redis_php_ddd/app/di-config.php ```

В нем описываются необходимы зависимости.
